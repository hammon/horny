# Horny
Server side file browser. Properties and Json editors.

### Build 
```
gradle buildJar
```
### Run 
```
cd build\libs
java -jar horny-1.0.jar -p &lt;PORT&gt; -r &lt;ROOT_PATH&gt;
````
